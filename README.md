# Interview Battlefield AI

Interview Battlefield AI is an AI-powered mock interview platform designed to simulate real technical interviews across multiple domains and difficulty levels.

The platform dynamically generates interview questions, evaluates user responses using AI, and generates detailed performance analytics reports.

---

## Live Demo

https://interview-battlefield-ai-439947078455.us-central1.run.app

---

## Features

- AI-generated technical interview questions
- Frontend, Backend, HR, and AI Engineer interview modes
- Easy, Medium, and Hard difficulty levels
- AI-powered answer evaluation system
- Performance analytics dashboard
- Expected answer comparison
- Strength and growth area analysis
- Interview history tracking
- PDF report generation
- Anti-cheating monitoring system
- Dockerized cloud deployment

## Project Workflow

1. User selects interview domain
2. User selects difficulty level
3. AI generates interview questions dynamically
4. User answers questions individually
5. AI evaluates performance
6. Analytics report is generated including:
   - Overall score
   - Performance analytics
   - Key strengths
   - Growth areas
   - Expected answers
   - Question breakdown
